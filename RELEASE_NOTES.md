# 0.13.0 发布说明

## 新增

- 新增保存搜索通知能力，支持通过 Webhook 和 Telegram Bot 接收搜索结果提醒。
- 新增 Telegram Bot 集成，支持机器人菜单、消息去重、账号绑定和按聊天会话订阅保存搜索。
- 新增资源 RSS 订阅，便于从外部阅读器或自动化流程跟踪资源更新。
- 新增已索引频道资源关联与频道清空操作，频道维护更直接。
- 新增初始设置频道搜索，提升大量频道场景下的选择效率。
- 新增宽屏内容模式和资源视频缩略图悬停预览，提升资源浏览体验。

## 优化

- 优化通知设置页面的交互与文案，并补充多语言展示。
- 启用监听时会同步近期历史，减少首次监听后的遗漏。
- Bot 通知现在会携带来源消息链接，便于回到 Telegram 查看上下文。
- 频道清空操作增加确认弹窗，降低误操作风险。
- 优化界面细节，提升列表和资源页面的可读性。

## 修复

- 修复未登录账号也能执行 Bot 命令的问题。
- 修复消息全文搜索查询转义不完整导致的异常或错误匹配。
- 修复管理员会话过期只在前端生效的问题，现在服务端会同步失效。
- 修复调度任务 panic 可能影响后续任务执行的问题。
- 修复首次设置相关接口缺少管理员校验的问题。
- 修复首次运行频道加载提示不清晰的问题。
