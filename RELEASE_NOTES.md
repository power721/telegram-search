# v0.15.1 发布说明

## 新增功能

- 移动端响应式支持：优化所有视图在移动设备上的显示效果
- 资源索引重建维护工具：提供索引维护和重建功能

## 性能优化

- 资源搜索性能优化：引入资源索引读模型，大幅提升公共搜索性能
- 修复资源 API 的 N+1 查询问题：使用批量获取优化文件加载
- 优化分页查询：使用快速 COUNT(DISTINCT) 提供准确的分页总数
- 禁用阻塞性的图片和视频缩略图生成

## 问题修复

- 修复管理员会话持久化
- 修复资源链接按 URL 索引的问题

## 其他改进

- 移除管理后台 API 调用中的 X-API-Key 认证头
- 优化 UI 界面
