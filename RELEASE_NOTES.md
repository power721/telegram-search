# 0.14.0 发布说明

## 新增

- 新增 Telethon session 登录能力，初始设置和账号页面都可以直接导入 Telethon session。
- 新增账号频道同步按钮，便于手动刷新账号已加入频道。
- 新增频道头像获取与展示，频道列表和相关页面更容易识别频道。
- 新增资源热门排序和趋势接口，支持按热度发现近期资源变化。

## 优化

- 优化频道与账号相关页面的展示细节，提升频道管理体验。
- 优化 Telegram Bot 相关处理流程，提升消息与链接展示的一致性。

## 修复

- 修复频道头像缓存无法在重新同步后正确更新的问题。
- 修复频道头像在部分 Telegram 响应中无法提取的问题。
- 修复私有频道标题链接中的频道 ID 格式不正确的问题。
